# Project Euler – Haskell Solutions

A collection of solutions to Project Euler problems implemented in Haskell, focused on developing problem-solving skills and mathematical reasoning through functional programming.

## Overview

This repository serves as a practical exploration of:

* Functional programming in Haskell
* Translating mathematical ideas into code
* Logical reasoning and problem decomposition
* Working with concepts from number theory, combinatorics, and discrete mathematics

## Highlights

* Clear and idiomatic Haskell code
* Emphasis on expressive solutions and functional design
* Variety of approaches reflecting different ways to model problems

## Structure

* Each problem is implemented as a separate module/file
* Some shared helper functions may be extracted where useful

## Usage

Solutions can be run using `ghci` or compiled with `ghc`.

## Disclaimer

This repository may contain full solutions to Project Euler problems and is intended for educational purposes.
